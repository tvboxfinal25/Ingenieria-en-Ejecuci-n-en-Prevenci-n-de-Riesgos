# Ingeniería en Ejecución en Prevención de Riesgos

App educativa gamificada sobre integridad y ética profesional para la carrera de Ingeniería en Ejecución en Prevención de Riesgos de la Universidad de Atacama, sede Vallenar.

## Características

- **Glassmorphism Design:** Interfaz moderna y elegante basada en el tema "Frosted Glass".
- **Gamificación:** Sistema de puntos, insignias y niveles de maestría.
- **Contenido Modular:** Módulos de teoría, actividades prácticas y evaluación final.
- **Generación de Reportes:** Exportación de resultados a PDF.
- **Responsivo:** Diseño adaptado para dispositivos móviles y escritorio.

## Tecnologías

- React 19
- Vite
- Tailwind CSS 4
- Lucide React (Iconos)
- Framer Motion (Animaciones)
- jsPDF (Reportes)
- Canvas Confetti (Efectos)

## Despliegue en Vercel

1. Sube este repositorio a GitHub.
2. Importa el proyecto en Vercel.
3. Configura las variables de entorno necesarias (ver `.env.example`).
4. Vercel detectará automáticamente la configuración de Vite.

## Desarrollo Local

```bash
npm install
npm run dev
```
